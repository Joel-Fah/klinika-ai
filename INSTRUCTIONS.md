## 0. Project Context & Problem Statement

### What we're building

**klinika_ai** is a Flutter application that demonstrates **Generative UI (GenUI)** — a paradigm where the AI model doesn't just respond with text, but actively constructs and renders real Flutter widgets in real time, based on a live conversation with the user.

This is not a chatbot. This is not a static form. This is an adaptive, agentic UI that thinks and builds itself.

### The local problem

Health facilities in Cameroon — from CHU Yaoundé to a rural health post in Mbalmayo — all face the same intake problem: **one-size-fits-all paper forms and rigid digital UIs** that ignore the patient in front of them. A child with a rash and a 70-year-old with chest pain should never see the same intake screen.

`klinika_ai` solves this by letting patients describe their situation in plain language (French, English, or Camfranglais), and having Gemini **build the right intake form dynamically**, on the fly, for that exact patient, at that exact moment. No hardcoded screens. No static dropdowns. A UI that earns its place by being genuinely useful.

### Workshop goal

Attendees should walk away understanding:
1. Why static UI is a ceiling, not a floor
2. How `genui` works: `Conversation`, `Catalog`, `Surface`, `DataModel`
3. How to wire Gemini into a real Flutter app via `firebase_ai`
4. How to create custom `CatalogItem`s that the AI can use
5. That GenUI isn't magic — it's architecture

## 10. Custom Catalog Items

These are the custom widgets the AI will use. Each one is a `CatalogItem` — a named widget with a JSON schema + a builder. Define them all in:

### File: `lib/features/intake/catalog/klinika_catalog.dart`

```dart
import 'package:flutter/material.dart';
import 'package:flutter_animate/flutter_animate.dart';
import 'package:gap/gap.dart';
import 'package:genui/genui.dart';
import 'package:json_schema_builder/json_schema_builder.dart';
import '../../../app/theme.dart';

// ─────────────────────────────────────────
// 1. TriageBanner
// Shown at top of a session surface.
// AI provides: severity (urgent|moderate|routine), message
// ─────────────────────────────────────────

final _triageBannerSchema = S.object(
  properties: {
    'severity': S.string(
      description: 'Urgency level: urgent | moderate | routine',
      enumValues: ['urgent', 'moderate', 'routine'],
    ),
    'message': S.string(
      description:
          'A short French message explaining the triage level (max 15 words)',
    ),
  },
  required: ['severity', 'message'],
);

CatalogItem buildTriageBannerItem() {
  return CatalogItem(
    name: 'TriageBanner',
    dataSchema: _triageBannerSchema,
    widgetBuilder: (context) {
      final severityNotifier = context.dataContext.subscribeToString(
        context.data['severity'] as Map<String, Object?>?,
      );
      final messageNotifier = context.dataContext.subscribeToString(
        context.data['message'] as Map<String, Object?>?,
      );

      return ValueListenableBuilder<String?>(
        valueListenable: severityNotifier,
        builder: (ctx, severity, _) {
          return ValueListenableBuilder<String?>(
            valueListenable: messageNotifier,
            builder: (ctx, message, _) {
              final color = switch (severity) {
                'urgent' => KlinikaPalette.urgent,
                'moderate' => KlinikaPalette.moderate,
                _ => KlinikaPalette.routine,
              };
              final label = switch (severity) {
                'urgent' => KlinikaStrings.urgent,
                'moderate' => KlinikaStrings.moderate,
                _ => KlinikaStrings.routine,
              };

              return Container(
                width: double.infinity,
                padding: const EdgeInsets.symmetric(
                  horizontal: KlinikaSpacing.md,
                  vertical: KlinikaSpacing.sm + 4,
                ),
                decoration: BoxDecoration(
                  color: color.withOpacity(0.12),
                  borderRadius: KlinikaRadius.md,
                  border: Border.all(color: color.withOpacity(0.4)),
                ),
                child: Row(
                  children: [
                    Container(
                      width: 10,
                      height: 10,
                      decoration: BoxDecoration(
                        color: color,
                        shape: BoxShape.circle,
                      ),
                    ),
                    const Gap(KlinikaSpacing.sm),
                    Text(
                      label,
                      style: TextStyle(
                        color: color,
                        fontSize: 12,
                        fontWeight: FontWeight.w800,
                        letterSpacing: 1.2,
                      ),
                    ),
                    const Gap(KlinikaSpacing.md),
                    Expanded(
                      child: Text(
                        message ?? '',
                        style: const TextStyle(
                          color: KlinikaPalette.snowWhite,
                          fontSize: 13,
                        ),
                      ),
                    ),
                  ],
                ),
              ).animate().fadeIn(duration: 400.ms).slideY(begin: -0.2);
            },
          );
        },
      );
    },
  );
}

// ─────────────────────────────────────────
// 2. DepartmentSelector
// AI suggests which department the patient should go to.
// AI provides: department (string), reason (string)
// ─────────────────────────────────────────

final _departmentSelectorSchema = S.object(
  properties: {
    'department': S.string(
      description:
          'Clinical department in French, e.g. Médecine Générale, Pédiatrie, Urgences, ORL',
    ),
    'reason': S.string(
      description: 'One-sentence reason in French for this department',

  },
  required: ['department'],

CatalogItem buildDepartmentSelectorItem() {
  return CatalogItem(
    name: 'DepartmentSelector',
    dataSchema: _departmentSelectorSchema,
    widgetBuilder: (context) {
      final deptNotifier = context.dataContext.subscribeToString(

      final reasonNotifier = context.dataContext.subscribeToString(
        context.data['reason'] as Map<String, Object?>?,

      return ValueListenableBuilder<String?>(
        valueListenable: deptNotifier,
        builder: (ctx, dept, _) {
          return ValueListenableBuilder<String?>(
            valueListenable: reasonNotifier,
            builder: (ctx, reason, _) {
              return Container(
                width: double.infinity,
                padding: const EdgeInsets.all(KlinikaSpacing.md),

                  color: KlinikaPalette.emeraldSurface,

                        color: KlinikaPalette.emerald,

                        letterSpacing: 1.1,

            },

// ─────────────────────────────────────────
// 3. SymptomChipGroup
// AI generates a group of tappable symptom chips for quick selection.
// AI provides: label, chips (array of strings)
// Note: chips are display-only in this version (logged to DataModel on tap)
// ─────────────────────────────────────────

final _symptomChipGroupSchema = S.object(
  properties: {
    'label': S.string(description: 'Section label in French, e.g. "Symptômes associés"'),
    'chips': S.array(
      items: S.string(description: 'A symptom chip label'),
      description: 'List of symptom options to display',
    ),
  },
  required: ['label', 'chips'],
);

CatalogItem buildSymptomChipGroupItem() {
  return CatalogItem(
    name: 'SymptomChipGroup',
    dataSchema: _symptomChipGroupSchema,
    widgetBuilder: (context) {
      final labelNotifier = context.dataContext.subscribeToString(
        context.data['label'] as Map<String, Object?>?,
      );

      // chips is an array — subscribe as a list
      final chips = (context.data['chips'] as List?)
              ?.map((e) => e.toString())
              .toList() ??
          [];

      return ValueListenableBuilder<String?>(
        valueListenable: labelNotifier,
        builder: (ctx, label, _) {
          return Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              if (label != null)
                Text(
                  label,
                  style: const TextStyle(
                    color: KlinikaPalette.mist,
                    fontSize: 13,
                    fontWeight: FontWeight.w600,
                  ),
                ),
              const Gap(KlinikaSpacing.sm),
              Wrap(
                spacing: KlinikaSpacing.sm,
                runSpacing: KlinikaSpacing.sm,
                children: chips
                    .map(
                      (chip) => ActionChip(
                        label: Text(chip),
                        onPressed: () {},
                        backgroundColor: KlinikaPalette.inkMid,
                        side: const BorderSide(color: KlinikaPalette.inkBorder),
                        labelStyle: const TextStyle(
                          color: KlinikaPalette.snowWhite,
                          fontSize: 13,
                        ),
                      ),
                    )
                    .toList(),
              ),
            ],
          ).animate().fadeIn(duration: 400.ms);
        },
      );
    },
  );
}

Catalog buildKlinikaCatalog() {

    buildTriageBannerItem(),
```

```dart
import 'package:firebase_ai/firebase_ai.dart';
import 'package:flutter/material.dart';
import 'package:genai_primitives/genai_primitives.dart';
import 'package:genui/genui.dart';
import 'package:get/get.dart';
import '../../../core/constants/strings.dart';
import '../catalog/klinika_catalog.dart';

class IntakeController extends GetxController {
  late final SurfaceController surfaceController;
  late final A2uiTransportAdapter transport;
  late final Conversation conversation;

  final textController = TextEditingController();
  final surfaceIds = <String>[].obs;
  final isLoading = false.obs;
  final hasStarted = false.obs;

  // Gemini model via Firebase AI (Vertex AI in Firebase)
  late final GenerativeModel _model;

  static const _systemPrompt = '''
You are a smart, empathetic patient intake assistant for a health facility in Yaoundé, Cameroon.
Patients may speak French, English, Camfranglais, or a mix. Always respond in the same language the patient uses.
Your job is to build an adaptive intake form using GenUI widgets — not return plain text.

RULES:
1. When a patient describes symptoms, always start by generating a TriageBanner with:
   - severity: "urgent" | "moderate" | "routine"
   - message: a short explanation in the patient's language

2. Then generate a DepartmentSelector with the most appropriate department.

3. Then generate a SymptomChipGroup with related symptoms they might confirm.

4. Then generate a form with relevant follow-up fields (using TextField, Select, etc).
   Only ask what is clinically relevant to what the patient described.
   Do not ask for information that is already obvious from their description.

5. Keep the conversation human. Don't be robotic.

6. Never generate the same form twice. Each response should adapt to what you now know.

IMPORTANT: Always generate at least one UI surface per response.
Never respond with plain text only — always build UI.
''';

  @override
  void onInit() {
    super.onInit();
    _initFirebaseAI();
    _initGenUI();
  }

  void _initFirebaseAI() {
    _model = FirebaseAI.googleAI().generativeModel(
      model: 'gemini-2.0-flash',
      systemInstruction: Content.system(_systemPrompt),
      generationConfig: GenerationConfig(
        temperature: 0.7,
        maxOutputTokens: 2048,
      ),
    );
  }

  void _initGenUI() {
    surfaceController = SurfaceController(
      catalogs: [buildKlinikaCatalog()],
    );

    transport = A2uiTransportAdapter(onSend: _onSendToGemini);

    conversation = Conversation(
      controller: surfaceController,
      transport: transport,
    );

    conversation.events.listen((event) {
      if (event is ConversationSurfaceAdded) {
        surfaceIds.add(event.surfaceId);
      } else if (event is ConversationSurfaceRemoved) {
        surfaceIds.remove(event.surfaceId);
      }
    });
  }

  Future<void> _onSendToGemini(ChatMessage message) async {
    isLoading.value = true;
    try {
      final stream = _model.generateContentStream([
        Content.text(message.content),
      ]);

      await for (final chunk in stream) {
        transport.addChunk(chunk.text ?? '');
      }
    } catch (e) {
      debugPrint('[klinika_ai] Gemini error: $e');
    } finally {
      isLoading.value = false;
    }
  }

  void sendMessage() {
    final text = textController.text.trim();
    if (text.isEmpty) return;

    hasStarted.value = true;
    conversation.sendRequest(ChatMessage.user(text));
    textController.clear();
  }

  void resetSession() {
    surfaceIds.clear();
    hasStarted.value = false;
    isLoading.value = false;
    textController.clear();
    // Re-init conversation for a fresh session
    conversation.dispose();
    transport.dispose();
    surfaceController.dispose();
    _initGenUI();
  }

  @override
  void onClose() {
    textController.dispose();
    conversation.dispose();
    transport.dispose();
    surfaceController.dispose();
    super.onClose();
  }
}
```
